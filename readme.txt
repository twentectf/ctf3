Step 1: Accessing robots.txt

Participants are directed to examine the source code where they'll find a clue in a comment guiding them to access the robots.txt file.
Accessing robots.txt reveals a secret HTML file accessible to public users.

Step 2: Manipulating the Cookie

Participants are instructed to inspect the cookie and decode the user value from Base64, using tools like CyberChef website.
Decoding the user value reveals it as "student".
Participants must then generate an encoded text equal to "admin" and edit the cookie with this new value.
Finally, they need to refresh the page to obtain the flag.
